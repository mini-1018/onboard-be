## 1. About
OnBoard는 개발자들이 코딩 과정에서 겪은 문제나 고민을 공유하고, 해결되지 않은 질문을 자유롭게 나눌 수 있는 커뮤니티입니다.
사이트 이름인 OnBoard는 게시판(Board)과 '승선'(On Board)의 중의적 의미를 담고 있으며, 돛의 모양은 태그를 형상화하였습니다.
